### Geometric Diagram QA
大致分为2步：1）将题干和几何示意图转换为形式描述。2）通过演绎推理，得到问题的答案。  
难点1：当示意图不符合比例时，题干与示意图的互补增强。  
难点2：演绎推理过程中的辅助线的做法及匹配的演绎推理。
